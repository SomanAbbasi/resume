# Soman Abbasi Portfolio

Welcome to my **React-based portfolio website**, built with **Vite** for fast development and smooth performance.  
Check it live here: [https://somanabbasi.vercel.app/](https://somanabbasi.vercel.app/)

---

## Features

- **Modern UI/UX:** Clean and responsive design built with React and Tailwind CSS.  
- **Tech Stack Display:** Highlights the technologies used for each project with icons.  
- **Project Showcase:** Interactive cards with descriptions, links to GitHub repos, and live demos.  
- **SPA Routing:** Smooth single-page navigation for a professional user experience.  
- **Continuous Updates:** Portfolio is actively updated with new projects and features.

---

## Tech Stack

- **Frontend:** React, Vite, Tailwind CSS  
- **Icons:** React Icons  
- **Hosting/Deployment:** Vercel (automatic GitHub deployment)  

---

## Getting Started

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/SomanAbbasi/portfolio.git
   cd portfolio
Install dependencies:

npm install


Start the development server:

npm run dev


Open your browser at the URL provided in the terminal (usually http://localhost:5173).
