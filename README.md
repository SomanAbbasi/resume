# Personal Developer Portfolio

A modern, responsive, and performance-optimized developer portfolio built with **Next.js**, showcasing my projects, skills, and experience as a Full-Stack Developer.

🌐 Live Site: https://somanabbasi.vercel.app/

---

##  Features

-  Modern UI with dark theme and gradient accents  
-  Fully responsive (mobile, tablet, desktop)  
-  Component-based architecture (Next.js App Router)  
-  Featured projects showcase with GitHub & live links  
-  Smooth animations and micro-interactions (Framer Motion)  
-  Clean and scalable code structure  
-  Optimized performance and fast load times  
-  SEO-friendly with proper metadata and OpenGraph support  

---

##  Tech Stack

- **Framework:** Next.js 14+ (App Router)  
- **Language:** TypeScript  
- **Styling:** Tailwind CSS  
- **Animations:** Framer Motion  
- **Icons:** Lucide React  
- **Deployment:** Vercel
